The folder "Main Study" contains the materials and data of the main study. 

The folder "Pilot Study of Exp1" contains the data and a summary of the results of the pilot study. 

The folder "Predictions step by step" contains a pdf file in which we formally derive the predictions for the 28 test cases of Experiment 1. 

The folder "Probability of singular causation vs. diagnostic probability for the test cases of Exp1" contains a supplementary file pitting the probability of singular causation against the diagnostic probability, for the 28 test cases of our Experiment 1.

The folder "R Code to generate odel predictions of Exp1" contains an R-Markdown (Rmd) File, that can be used to generate the predictions.
